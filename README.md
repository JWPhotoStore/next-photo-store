# next-photo-store 🎞️

Online photo store, branded as "Kushi Photos". This is a fullstack project written in TypeScript.

## Technologies Used

- Next.js
- React
- Redux Toolkit + RTK Query
- Prisma
- PostgreSQL
- Stripe API
